# portfolio

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production
$ npm run build

# preview production build locally
$ npm run preview

# generate static project (if using Nuxt 4's static target)
$ npm run generate
```
